The Business card extracter could be used by sales team , they often go to exhibitions, presentation and so. They receieve business cards from customer/clients , later they need to feed into a csv file manually 
which is time consuming. THIS APP HELPS TO DO THAT.

Install all the dependencies listed in the requirements.txt

Download tesseract

Most important : Check your systems path correctly, Download "python -m spacy en_core_web_sm", this is a pretrained model.

Follow the folder structure to avoid conflicts.
